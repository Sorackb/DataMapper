DataMapper
==========
This framwork can be configured to access informations from database based on XML configuration's file.

### Deploy structure

```
DataMapper/
├── DataMapper.jar
├── config.ini
```

### config.ini structure

- basic.mappingfolder
- connection.host (optional)
  - `(default) 127.0.0.1`
- connection.database
- connection.user
- connection.password
